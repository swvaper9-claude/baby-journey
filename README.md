# 🌸 Baby Journey

임산부와 가족을 위한 임신 여정 공유 앱

## 📁 파일 구조

```
baby-journey/
├── index.html
├── package.json
├── vite.config.js
├── .gitignore
└── src/
    ├── main.jsx
    └── App.jsx
```

## 🚀 Vercel 배포 방법

1. GitHub에 이 폴더 전체를 저장소로 업로드
2. vercel.com → New Project → GitHub 저장소 선택
3. Framework: **Vite** 선택
4. Deploy 클릭
5. 완성된 URL을 카톡으로 공유!

## 📱 사용법

- **엄마**: ✏️ 편집 탭 → PIN 입력(기본: 1234) → 정보 입력 → 저장
- **가족**: 링크 열면 바로 최신 정보 확인 가능

## ⚠️ 참고

현재 버전은 각 기기의 로컬 저장소를 사용합니다.
완전한 실시간 공유를 위해서는 Firebase 연동이 필요합니다.
